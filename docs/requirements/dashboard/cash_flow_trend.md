# 功能需求: 现金流走势环比 (Cash Flow Trend MoM)

> **所属模块**: Dashboard
> **优先级**: P1

## 1. 需求描述
总经理需要直观对比本月与上月同期的现金流（余额）走势，以快速判断资金消耗速度是否异常。

## 2. 功能清单 (Checklist)
- [ ] **可视化图表**: 
  - 使用面积图 (Area Chart) 展示。
  - **蓝色实线**: 本月每日余额趋势。
  - **灰色虚线**: 上月同期每日余额趋势。
  - **弱化金额**: 隐藏 Y 轴具体数值，仅展示趋势形态。
- [ ] **交互 (Tooltip)**:
  - 点击/悬浮图表节点显示详细对比数据。
  - 自动计算并高亮显示**差额**与**涨跌幅百分比**（如 +5.2%）。
- [ ] **数据范围**:
  - 横轴: 1日 ~ 30/31日。
- [ ] **摘要展示**:
  - 在图表标题栏旁显示当前的整体环比涨跌幅。
- [ ] **入口**: 
  - 不显示具体金额，仅展示走势。
  - 点击跳转至“资金流水”详情页。
