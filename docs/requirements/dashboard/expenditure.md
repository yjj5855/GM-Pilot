# 功能需求: 支出构成分析 (Expenditure Breakdown)

> **所属模块**: Dashboard
> **优先级**: P1

## 1. 需求描述
让管理者直观感知本月钱花哪儿了，特别是刚性的人力成本与税费。

## 2. 功能清单 (Checklist)
- [ ] **可视化**: 仅展示饼图，去除总金额的大数展示。
- [ ] **饼图可视化**: 
  - 维度: 工资薪金、五险一金、纳税总额、外包服务费。
  - 交互: 点击扇区或图例可查看具体金额。
- [ ] **跳转入口**: 点击箭头跳转至“财税报表”详情页。

## 3. UI 细节
- 配色需区分“成本”(蓝系) 与 “税费”(橙/黄系)。
