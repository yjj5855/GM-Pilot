# 功能需求: 交付进度时间轴 (Timeline)

> **所属模块**: Dashboard
> **优先级**: P0

## 1. 需求描述
按时间顺序展示本月关键业务节点，让用户清晰了解当前进度。

## 2. 功能清单 (Checklist)
- [ ] **节点状态展示**: 
  - Completed (绿): 已完成。
  - Active (蓝): 进行中，显示当前处理的具体细节。
  - Pending (灰): 待开始。
  - Risk (橙): 存在异常或风险。
- [ ] **日期显示**:
  - 每个节点需显示具体日期（如 12月25日）。
  - **五险一金**: 固定为每月 15-25 日。
- [ ] **节点内容更新**:
  - **五险一金缴纳**: 需展示五险缴纳金额、一金缴纳金额及环比趋势。
  - **增减员与招退工**: 需展示本月增员人数、本月减员人数。
  - **政府补助申请**: 列出具体申请清单（如稳岗补贴），每行后面以标签 (Label) 形式体现状态。
- [ ] **交互更新**:
  - 所有节点（包括工资、五险一金、补助等）均以 **Link 形式** 提供点击入口，跳转查看详情。
  - 去除原有的显式 Action Button（如“去确认”按钮）。
- [ ] **默认折叠**: 仅展示 Active (进行中) 或 Risk (风险) 节点，节省首屏空间。
- [ ] **展开全部**: 点击头部标题或底部“查看全部”按钮展开完整时间轴。
- [ ] **进度概览**: 标题栏右侧显示整体进度百分比 (如 25%)。

## 3. 数据来源
- 外包服务商履约系统状态同步。
